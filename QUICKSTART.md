# 🚀 Quick Start Guide

## Start Development Server

### Option 1: Double-click
```
start.bat
```

### Option 2: Command line
```bash
npm run dev
```

## Access the Application

Open your browser to: **http://localhost:5173**

## What You'll See

### Single-Page AI Workspace
- **Left Sidebar**: Logo, New Chat, History, Tools, Files, Settings
- **Main Area**: Welcome screen with animated icon and suggestions
- **Bottom Input**: Floating message input with action menu
- **Background**: Animated star field + floating cosmic orbs

## How to Use

1. **Type a message** in the input box at bottom
2. **Press Enter** or click send button
3. **See AI response** appear with smooth animation
4. **Click + button** to open action menu:
   - Upload Repository
   - Build Website
   - Analyze Code
   - Generate Architecture
5. **Start new chat** by clicking "New Chat" in sidebar
6. **View history** in the sidebar chat list
